Soft Robotics

Build your own Soft Robotic

  • Soft Robotics are mobile components which can be moved for example by air pressure. In Care fabrication it can be used for assistive fingers or like the picture in the showcase for machines to pick materials.
    We took the bodyscan of Leyla which also can be found in the Assignment of 3D scanning and wanted to build an breath assistant which should pressure the lungs for helping to exhale and same the other way around.

    The scan building the channels which should be bandable

    this pattern should be cutted into breadpaper because the both sides later on will be cutted with hot gluing synthetics

    The vinyl cutter has a knife which can be instructed with this turn knob.

    both cutted sides are now placed on each other with the bread paper in between. On the area where the breadpaper is, later on the air can go trough.(it will not be glued together)

    This machine works like a big hot iron. It only takes a few seconds and the whole robotic is glued.
